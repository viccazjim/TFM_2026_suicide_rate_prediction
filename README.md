# TFM 2026 — Predictive Analysis of Suicide Rates in the EU

Master's thesis (TFM) project: predicting suicide rates across EU countries from socioeconomic determinants and mental health prevalence, using a granular, production-style pipeline (`src/` modules + notebooks).

## Data sources

| Source | What it provides | Access |
|---|---|---|
| [IHME Global Burden of Disease](https://www.healthdata.org/) | Mental health / neurological disorder prevalence, per country/year/cause | Local CSV export (`data/raw/`) |
| World Bank API | GDP per capita, unemployment, health expenditure, population, Gini index, urban population %, physicians per 1,000, internet users % | Live API call, no key required |
| WHO Global Health Observatory API | Suicide rate per 100,000 inhabitants (target variable) | Live API call, no key required |

Scope: 27 EU member states, years 2000–2021 (`df_development`, labeled) plus 2022–2023 (`df_real_world`, unlabeled — WHO has not published suicide rates for these years yet).

## Pipeline

Run the notebooks in order — each one reads what the previous one wrote to `data/processed/`:

1. **`01_data_loading_cleaning.ipynb`** — fetches and merges the three sources, imputes missing values (linear interpolation for interior gaps, OLS trend extrapolation for boundary gaps), saves `df_development.csv` and `df_real_world.csv`.
2. **`02_eda.ipynb`** — suicide rate trends by country/region, distribution and outlier analysis, VIF-based multicollinearity check (drops `Eating disorders`, VIF = 12.3), overwrites `df_development.csv` with the cleaned feature set.
3. **`03_models.ipynb`** — trains Ridge, Lasso, SVR, Random Forest, XGBoost and CatBoost under two split strategies:
   - **Option A (geographical):** 70/15/15 split by country — tests generalization to unseen countries.
   - **Option B (time):** 70/15/15 split by year — tests generalization to future years.

   Results are saved to `outputs/tables/`. The notebook also includes result diagnostics (actual vs predicted, residuals, error by year/country) and a SHAP-based interpretability section for the best-performing model.

## Results summary

From the result tables currently saved in `outputs/tables/`:

- **Option A (geographical split):** all six models reach a positive Test R² (0.08–0.44), but five of six drop to a negative R² on Validation — the positive Test score does not transfer to genuinely unseen countries. SVR is the only model that holds up (R² 0.44 → 0.13).
- **Option B (time split):** stronger and more stable — Test R² 0.18–0.88, Validation R² 0.08–0.63. CatBoost and SVR are the most consistent performers across both.

This suggests the available socioeconomic/mental-health indicators carry real predictive signal for near-future rates within known countries, but do not capture whatever country-specific structural factors (healthcare system, culture, history) drive cross-country differences.

SHAP analysis on the best Option B model (CatBoost) shows `Alcohol use disorders` as by far the single most influential predictor — its mean absolute SHAP value (~2.9) is roughly 3x the next-ranked feature (`Attention-deficit/hyperactivity disorder`, ~0.98). Classic socioeconomic determinants (`Unemployment rate`, `Health expenditure`) rank lowest. This does not necessarily mean unemployment is unimportant for suicide risk in general — it likely reflects that, within Option B's fixed set of countries, unemployment varies more *across* countries than *within* a country over time, and Option B only ever asks the model to explain variation over time for countries it has already seen.

Validation-set error is not flat across years: mean absolute error rises from ~2.0 (2018) to ~2.9 (2021), i.e. the model degrades specifically on the two years that include the COVID-19 pandemic (2020–2021), which the training period (2000–2014) obviously could not have seen. Part of the Option B Test/Val gap reported above is attributable to this, not only to generic time-distance decay.
